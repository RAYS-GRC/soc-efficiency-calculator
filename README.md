# SOC Efficacy Calculator

React + Vite + TypeScript single-page app implementing a SEBI CSCRF Annexure-N style
SOC efficacy calculator with:

- Multi-step wizard (ORG, C, Y, P, H, E, Summary)
- Dark purple UI
- Input validation
- Domain scores + unique insights
- Printable report (use browser Print -> Save as PDF)

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
