# SOC Efficacy Calculator

React + Vite + TypeScript single-page app implementing a SEBI CSCRF Annexure-N style
SOC efficacy calculator with:

- Multi-step wizard (ORG, C, Y, P, H, E, Summary)
- Dark purple UI, mobile-friendly layout
- Input validation and domain-specific insights
- Domain scores + unique explanations (what each score means)
- Printable report (use browser Print -> Save as PDF)
- All calculations run **locally in the browser** â€“ no data is sent to any server.

> Open-source contribution by **Rajesh Yalavaguli Seetharamarao** to the GRC community.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
